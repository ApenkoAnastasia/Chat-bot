## Usage

In CLI start program.

Clone repository:
```commandline
git clone https://github.com/...
```

Create and activate a virtual environment:
```sh
$ python3 -m venv venv
$ . venv/bin/activate
```

To deactivate virtual environment:
```sh
$ deactivate
```
